# lita-latex

A [Lita][1] handler for rendering LaTeX.

## Installation

Add lita-latex to your Lita instance's Gemfile:

```ruby
gem 'lita-latex'
```

## Usage

zacstewart: Lita: latex me \Sigma

Lita: ![\Sigma](http://chart.apis.google.com/chart?cht=tx&chl=\Sigma)
